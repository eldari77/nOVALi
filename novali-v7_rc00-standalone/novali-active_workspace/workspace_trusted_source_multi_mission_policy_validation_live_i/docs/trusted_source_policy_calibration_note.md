# Trusted-Source Policy Calibration

Compared conservative review-first and permissive bounded trusted-source policies across readiness, packaging/integrity, and documentation/completeness missions. Readiness remained the only mission class that justified bounded external escalation under the permissive policy, while packaging and documentation continued to prefer local-only execution under both policies because local evidence stayed sufficient and cheaper.

Validated mission classes: readiness, packaging/integrity, documentation/completeness.
