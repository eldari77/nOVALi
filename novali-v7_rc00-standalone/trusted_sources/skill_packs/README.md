# Internal Skill Packs

These skill packs are packaged bounded execution helpers for successor-quality
work inside the active workspace.

They are distinct from `trusted_sources/knowledge_packs/`:

- knowledge packs are read-only planning and review evidence
- skill packs are bounded execution capabilities

Skill packs do not grant permissions, do not override governance authority, and
do not mutate protected surfaces.

Current bounded successor-quality skill packs:

- `successor_workspace_review_pack_v1.json`
- `successor_test_strengthening_pack_v1.json`
- `successor_manifest_quality_pack_v1.json`
- `successor_docs_readiness_pack_v1.json`
- `successor_artifact_index_consistency_pack_v1.json`
- `successor_handoff_completeness_pack_v1.json`
- `trusted_reference_inventory_integration_pack_v1.json`

These packs are intended to be sequenced conservatively through the successor
quality roadmap and composite evaluation artifacts:

- `artifacts/successor_quality_roadmap_latest.json`
- `artifacts/successor_quality_priority_matrix_latest.json`
- `artifacts/successor_quality_composite_evaluation_latest.json`
- `artifacts/successor_quality_next_pack_plan_latest.json`
