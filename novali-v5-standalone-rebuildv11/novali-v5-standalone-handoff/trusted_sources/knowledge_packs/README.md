# Internal Knowledge Packs

These knowledge packs are packaged read-only planning and review evidence for
governed continuation quality.

They define conservative bounded completion and continuation criteria inside the
active workspace only. They are evidence inputs, not governance authority.

Current packaged packs:

- `successor_completion_knowledge_pack_v1.json`
  - bounded successor-completion rubric
- `workspace_continuation_knowledge_pack_v1.json`
  - next-cycle derivation rubric for planning and implementation
- `successor_promotion_review_knowledge_pack_v1.json`
  - review-and-promotion rubric plus bounded next-objective templates
